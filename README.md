This project is a simple demonstration on how to implement the A* algorithm in
Python using pygame.

## Requirements

* python>=3.7

### Dependencies

* pygame
* venv (completely optional. only if you want the libraries used in this program
  to be apart from the system libraries).

## How to run the project?

1. clone the project

    `git clone https://github.com/smolBlackCat/python-pathfinding.git`

2. enter the project directory

    `cd python-pathfinding`

3. create a python virtual environment (optional)

    `python3 -m venv venv`

4. install pygame using pip

    `pip install pygame`

5. enter the source code directory and run it

    ```sh
    cd src/
    python3 -m pathfinding.main_game
    ```
